# 🚀 빠른 시작 가이드

Flask 이미지 보정 애플리케이션을 빠르게 시작하는 방법입니다.

## 방법 1: 자동 실행 스크립트 (권장)

```bash
./run.sh
```

이 스크립트는 자동으로:
- 가상환경 생성 (없는 경우)
- 필요한 패키지 설치
- Flask 애플리케이션 실행

## 방법 2: 수동 실행

### 1단계: 가상환경 생성 및 활성화

```bash
python3 -m venv venv
source venv/bin/activate
```

### 2단계: 패키지 설치

```bash
pip install -r requirements.txt
```

### 3단계: 애플리케이션 실행

```bash
python app.py
```

## 접속하기

브라우저를 열고 다음 주소로 접속하세요:

```
http://localhost:5000
```

## 사용 방법

### 1. 이미지 업로드
- 화면의 업로드 영역을 클릭하거나
- 이미지 파일을 드래그 앤 드롭

### 2. 필터 적용
슬라이더를 조정하여 다음 효과를 적용:
- **밝기**: 이미지를 밝게 또는 어둡게
- **명암**: 대비를 높이거나 낮춤
- **채도**: 색상의 강도 조정
- **선명도**: 이미지의 날카로움 조정

### 3. 색 조정
고급 색상 조정:
- **색온도**: 따뜻한 톤 ↔ 차가운 톤
- **색조**: 색상 회전
- **HSL 채도**: 세밀한 채도 조정
- **명도**: 밝기 조정

### 4. 카메라 프로필
전문 카메라의 색감을 재현:
- **ARRI Alexa**: 자연스러운 색상
- **RED**: 선명하고 강렬한 색상
- **Canon**: 따뜻한 톤
- **Sony**: 균형잡힌 색상
- **Blackmagic**: 플랫한 프로필

### 5. 저장
- **다운로드** 버튼으로 보정된 이미지 저장
- **초기화** 버튼으로 모든 조정 리셋

## 팁

## 문제 해결

### 포트가 이미 사용 중인 경우

`app.py` 파일의 마지막 줄을 수정:

```python
app.run(debug=True, host='0.0.0.0', port=5001)  # 다른 포트 번호 사용
```

### 패키지 설치 오류

시스템에 Python 3.8 이상이 설치되어 있는지 확인:

```bash
python3 --version
```

### OpenCV 오류

OpenCV 관련 오류가 발생하면:

```bash
pip uninstall opencv-python
pip install opencv-python-headless
```
